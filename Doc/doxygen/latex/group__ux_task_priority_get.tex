INCLUDE\+\_\+ux\+Task\+Priority\+Get must be defined as 1 for this function to be available. See the configuration section for more information.

Obtain the priority of any task.


\begin{DoxyParams}{Parameters}
{\em x\+Task} & Handle of the task to be queried. Passing a NULL handle results in the priority of the calling task being returned.\\
\hline
\end{DoxyParams}
\begin{DoxyReturn}{Returns}

\end{DoxyReturn}
