\doxysubsubsection{Detailed Description}
Spline interpolation is a method of interpolation where the interpolant is a piecewise-\/defined polynomial called "{}spline"{}.

\begin{DoxyParagraph}{Introduction}

\end{DoxyParagraph}
Given a function f defined on the interval \mbox{[}a,b\mbox{]}, a set of n nodes x(i) where a=x(1)\texorpdfstring{$<$}{<}x(2)\texorpdfstring{$<$}{<}...\texorpdfstring{$<$}{<}x(n)=b and a set of n values y(i) = f(x(i)), a cubic spline interpolant \doxylink{arm__bayes__example__f32_8c_ad5af783c9e1eb1514dec0a59dac35eb7}{S(x)} is defined as\+:


\begin{DoxyPre}
         S1(x)       x(1) < x < x(2)
 \doxylink{arm__bayes__example__f32_8c_ad5af783c9e1eb1514dec0a59dac35eb7}{S(x)} =   ...         
         Sn-\/1(x)   x(n-\/1) < x < x(n)
 \end{DoxyPre}


where


\begin{DoxyPre} 
 Si(x) = a\_i+b\_i(x-\/xi)+c\_i(x-\/xi)^2+d\_i(x-\/xi)^3    i=1, ..., n-\/1
 \end{DoxyPre}


\begin{DoxyParagraph}{Algorithm}

\end{DoxyParagraph}
Having defined h(i) = x(i+1) -\/ x(i)


\begin{DoxyPre}
 h(i-\/1)c(i-\/1)+2[h(i-\/1)+h(i)]c(i)+h(i)c(i+1) = 3/h(i)*[a(i+1)-\/a(i)]-\/3/h(i-\/1)*[a(i)-\/a(i-\/1)]    i=2, ..., n-\/1
 \end{DoxyPre}


It is possible to write the previous conditions in matrix form (Ax=B). In order to solve the system two boundary conidtions are needed.
\begin{DoxyItemize}
\item Natural spline\+: S1\textquotesingle{}\textquotesingle{}(x1)=2\texorpdfstring{$\ast$}{*}c(1)=0 ; Sn\textquotesingle{}\textquotesingle{}(xn)=2\texorpdfstring{$\ast$}{*}c(n)=0 In matrix form\+:
\end{DoxyItemize}


\begin{DoxyPre}
 |  1        0         0  ...    0         0           0     ||  c(1)  | |                        0                        |
 | h(0) 2[h(0)+h(1)] h(1) ...    0         0           0     ||  c(2)  | |      3/h(2)*[a(3)-\/a(2)]-\/3/h(1)*[a(2)-\/a(1)]      |
 | ...      ...       ... ...   ...       ...         ...    ||  ...   |=|                       ...                       |
 |  0        0         0  ... h(n-\/2) 2[h(n-\/2)+h(n-\/1)] h(n-\/1) || c(n-\/1) | | 3/h(n-\/1)*[a(n)-\/a(n-\/1)]-\/3/h(n-\/2)*[a(n-\/1)-\/a(n-\/2)] |
 |  0        0         0  ...    0         0           1     ||  c(n)  | |                        0                        |
 \end{DoxyPre}



\begin{DoxyItemize}
\item Parabolic runout spline\+: S1\textquotesingle{}\textquotesingle{}(x1)=2\texorpdfstring{$\ast$}{*}c(1)=S2\textquotesingle{}\textquotesingle{}(x2)=2\texorpdfstring{$\ast$}{*}c(2) ; Sn-\/1\textquotesingle{}\textquotesingle{}(xn-\/1)=2\texorpdfstring{$\ast$}{*}c(n-\/1)=Sn\textquotesingle{}\textquotesingle{}(xn)=2\texorpdfstring{$\ast$}{*}c(n) In matrix form\+:
\end{DoxyItemize}


\begin{DoxyPre}
 |  1       -\/1         0  ...    0         0           0     ||  c(1)  | |                        0                        |
 | h(0) 2[h(0)+h(1)] h(1) ...    0         0           0     ||  c(2)  | |      3/h(2)*[a(3)-\/a(2)]-\/3/h(1)*[a(2)-\/a(1)]      |
 | ...      ...       ... ...   ...       ...         ...    ||  ...   |=|                       ...                       |
 |  0        0         0  ... h(n-\/2) 2[h(n-\/2)+h(n-\/1)] h(n-\/1) || c(n-\/1) | | 3/h(n-\/1)*[a(n)-\/a(n-\/1)]-\/3/h(n-\/2)*[a(n-\/1)-\/a(n-\/2)] |
 |  0        0         0  ...    0        -\/1           1     ||  c(n)  | |                        0                        |
 \end{DoxyPre}


A is a tridiagonal matrix (a band matrix of bandwidth 3) of size N=n+1. The factorization algorithms (A=LU) can be simplified considerably because a large number of zeros appear in regular patterns. The Crout method has been used\+: 1) Solve LZ=B


\begin{DoxyPre}
 u(1,2) = A(1,2)/A(1,1)
 z(1)   = B(1)/l(11)

 FOR i=2, ..., N-\/1
   l(i,i)   = A(i,i)-\/A(i,i-\/1)u(i-\/1,i)
   u(i,i+1) = a(i,i+1)/l(i,i)
   z(i)     = [B(i)-\/A(i,i-\/1)z(i-\/1)]/l(i,i)

 l(N,N) = A(N,N)-\/A(N,N-\/1)u(N-\/1,N)
 z(N)   = [B(N)-\/A(N,N-\/1)z(N-\/1)]/l(N,N)
 \end{DoxyPre}


2) Solve UX=Z


\begin{DoxyPre}
 c(N)=z(N)

 FOR i=N-\/1, ..., 1
   c(i)=z(i)-\/u(i,i+1)c(i+1) 
 \end{DoxyPre}


c(i) for i=1, ..., n-\/1 are needed to compute the n-\/1 polynomials. b(i) and d(i) are computed as\+:
\begin{DoxyItemize}
\item b(i) = \mbox{[}y(i+1)-\/y(i)\mbox{]}/h(i)-\/h(i)\texorpdfstring{$\ast$}{*}\mbox{[}c(i+1)+2\texorpdfstring{$\ast$}{*}c(i)\mbox{]}/3
\item d(i) = \mbox{[}c(i+1)-\/c(i)\mbox{]}/\mbox{[}3\texorpdfstring{$\ast$}{*}h(i)\mbox{]} Moreover, a(i)=y(i).
\end{DoxyItemize}

\begin{DoxyParagraph}{Behaviour outside the given intervals}

\end{DoxyParagraph}
It is possible to compute the interpolated vector for x values outside the input range (xq\texorpdfstring{$<$}{<}x(1); xq\texorpdfstring{$>$}{>}x(n)). The coefficients used to compute the y values for xq$<$x(1) are going to be the ones used for the first interval, while for xq$>$x(n) the coefficients used for the last interval.

\begin{DoxyParagraph}{Initialization function}

\end{DoxyParagraph}
The initialization function takes as input two arrays that the user has to allocate\+: {\ttfamily coeffs} will contain the b, c, and d coefficients for the (n-\/1) intervals (n is the number of known points), hence its size must be 3\texorpdfstring{$\ast$}{*}(n-\/1); {\ttfamily temp\+Buffer} is temporally used for internal computations and its size is n+n-\/1.

\begin{DoxyParagraph}{}

\end{DoxyParagraph}
The x input array must be strictly sorted in ascending order and it must not contain twice the same value (x(i)\texorpdfstring{$<$}{<}x(i+1)).
