\doxysubsection{Detailed Description}
Provides definitions about the hardware and configuration of the Debug Unit.

This information includes\+:
\begin{DoxyItemize}
\item Definition of Cortex-\/M processor parameters used in CMSIS-\/\+DAP Debug Unit.
\item Debug Unit Identification strings (Vendor, Product, Serial Number).
\item Debug Unit communication packet size.
\item Debug Access Port supported modes and settings (JTAG/\+SWD and SWO).
\item Optional information about a connected Target Device (for Evaluation Boards). 
\end{DoxyItemize}
