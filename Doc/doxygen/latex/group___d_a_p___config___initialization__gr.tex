Setup of the Debug Unit I/O pins and LEDs (called when Debug Unit is initialized). This function performs the initialization of the CMSIS-\/\+DAP Hardware I/O Pins and the Status LEDs. In detail the operation of Hardware I/O and LED pins are enabled and set\+:
\begin{DoxyItemize}
\item I/O clock system enabled.
\item all I/O pins\+: input buffer enabled, output pins are set to HighZ mode.
\item for n\+TRST, n\+RESET a weak pull-\/up (if available) is enabled.
\item LED output pins are enabled and LEDs are turned off. 
\end{DoxyItemize}
