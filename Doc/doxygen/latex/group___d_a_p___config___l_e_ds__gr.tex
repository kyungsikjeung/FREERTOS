\doxysubsection{Detailed Description}
CMSIS-\/\+DAP Hardware may provide LEDs that indicate the status of the CMSIS-\/\+DAP Debug Unit.

It is recommended to provide the following LEDs for status indication\+:
\begin{DoxyItemize}
\item Connect LED\+: is active when the DAP hardware is connected to a debugger.
\item Running LED\+: is active when the debugger has put the target device into running state. 
\end{DoxyItemize}
