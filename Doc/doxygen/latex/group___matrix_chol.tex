\doxysubsubsection{Detailed Description}
Computes the Cholesky or LDL\texorpdfstring{$^\wedge$}{\string^}t decomposition of a matrix.

If the input matrix does not have a decomposition, then the algorithm terminates and returns error status ARM\+\_\+\+MATH\+\_\+\+DECOMPOSITION\+\_\+\+FAILURE.
