\doxysubsection{Detailed Description}
Access function for Test Domain Timer.

The value of the Test Domain Timer in the Debug Unit is returned by the function \doxylink{group___d_a_p___config___timestamp__gr_gaf9bdc40d3a256fc2cc4d26b295993d9c}{TIMESTAMP\+\_\+\+GET}. By default, the DWT timer is used. The frequency of this timer is configured with \doxylink{group___d_a_p___config___debug__gr_ga698a5a4ebd84335134a35e74848d2734}{TIMESTAMP\+\_\+\+CLOCK}.
